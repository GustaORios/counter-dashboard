# Counter Dashboard

## Objective
Practice React components, state management, and lifting state up.

## Description
A simple React app with two counters.  
Each counter can increase or decrease its value.  
The total count is shown in the parent component.  
There’s also a **Reset All** button to set all counters to zero.  
Each counter shows the **last updated time**.

## Components
- **App.js** – renders the Dashboard.  
- **Dashboard.js** – stores all counters’ state and the total.  
- **Counter.js** – displays and controls each counter.
